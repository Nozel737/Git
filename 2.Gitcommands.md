setup username: git config --global user.name "name"

setup email:git config --global user.email "myemail"

create default branch name: git config --global init.default branch main

create a git repository: git init

display all contents:ls

add current changes to study area: git add . , git add index.html(paticular files)

shows files to be committed to a git repository: git status(show what is suppose to be committed)

pushing infos to the final repository or server: git commit --amend

change editor: git config --global core.editor

to check who change anything on the site/app: git log

to show who did what, where, when and why: git blame

the new changes added that are not yet added to the stagging area: git diff or git status

create a branch:git branch main

give a name to a branch: git branch

give a new name to an existing branch:git branch main

takes another name: git switch

"main" is the new branch name: git switch main

create a new branch name: git checkout -b boss/master/king

delete any branch: git branch -d

connect your local repository to the remote repository(online): git remote add origin link from github

to push your informations from the local folder to the repository on github: git push -u origin main or git push -f origin main
